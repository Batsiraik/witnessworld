# Typography

## Admin Website

Current font: Montserrat

Previous font before the Montserrat update: Inter

Fallback stack: `system-ui, sans-serif`

To revert, update `admin/partials/head.php` by changing the Google Fonts import back to Inter and the Tailwind `fontFamily.sans` value back to `['Inter', 'system-ui', 'sans-serif']`.
